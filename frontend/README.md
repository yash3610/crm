# CRM Frontend

Standard React JavaScript single-page application built with Vite.

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Structure

- `src/main.jsx` mounts the React application.
- `src/App.jsx` defines the application routes and shared layout.
- `src/pages` contains page components.
- `src/components` contains shared UI and layout components.
- `src/styles.css` contains global styles.
